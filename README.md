# ☕ Java 8 Features - Hands-On Examples

A collection of practical Java 8 examples demonstrating Functional Programming concepts, Lambda Expressions, Streams API, Method References, Optional, Date & Time API, and Functional Interfaces.

## 🚀 Overview

This repository contains hands-on implementations of major Java 8 features with simple and easy-to-understand examples. It is designed for beginners and interview preparation, helping developers understand modern Java programming concepts through code.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/java8-features-hands-on.git
```

2. Open the project in IntelliJ IDEA.

3. Configure JDK 8 or higher.

4. Run individual examples from the respective packages.
